Work

Aarya in Images
A collection of 256 images that shaped my identity, from my photo albums thus far.

Research

I’ve learnt over the years that when it comes to projects like this, I tend to be able to ideate better if I establish a methodical way of reaching the end goal. So I started this exercise by first brainstorming a couple categories of things that have heavily impacted my life. I don’t want to list the categories here because I want whoever views my project to be able to identify those themes themselves, but if you ask me, I will tell you for sure!

From there, I began inputting photos from my photo albums into an airtable base, along with a title, source, and description of why I chose the photo.

After that, all I had to do was configure the API and design the site!

Reflection

When I started the exercise, I thought it would be so easy for me to find those 256 photos, but since one of the parameters I had set for myself was that each one needed a description, it became a lot more challenging for me to find photos. As I was collecting them, I realised how many memories remained for which I had lost the photos, and vice versa, and it became quite an emotional experience to write out why things that felt insignificant in the moment meant so much to me now.

I feel like the biggest takeaway from this project though were the new skills I learnt while coding it. Airtable has changed their API access method from API keys to personal tokens and so I had to relearn how to write the necessary functions to call the API myself. I also learned how to write promises and async functions in Javascript, and iterated on a “draggable” function that I have used before.

Overall, I estimate that I spent about 12 hours this week working on thesis ideas and this exercise.
